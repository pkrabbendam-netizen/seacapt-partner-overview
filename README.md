# SEACAPT Partner Overview

This is a public-safe, non-technical Streamlit overview of the SEACAPT concept for external partner conversations.

It is intentionally simple. It explains the early-stage R&D idea in plain language and does not include private technical materials, modelling files, funding details, hidden project data, or specialist review packs.

## Streamlit Cloud

Use this entry point:

```text
dashboard_simple.py
```

## Local run

```bash
streamlit run dashboard_simple.py
```
